📲 インストール手順

  1. 上の「APK をダウンロード」ボタンをタップ
  2. Chrome がダウンロード警告を出したら「ダウンロードを続行」
  3. 通知バーまたは「ファイル」アプリからダウンロードした APK を開く
  4. 初回は「不明なアプリのインストール」の許可を求められるので、Chrome（やファイルアプリ）に許可を与える
  5. 「インストール」をタップ
  6. 完了後、ホームから「プリズマ☆リンク」を起動
⚠ 前バージョン（旧名 "MyApp" / パッケージ com.example.myapp）が 入っている場合は、設定 → アプリ → 「MyApp」をアンインストールしてからご利用ください。 パッケージ ID が変わっているため、本バージョンは新規アプリとして並んで入ります。
ℹ 「アプリがインストールされていません」エラーが出たら:
① 既存の同名アプリを設定からアンインストール
② ストレージ空き容量を確認（最低 30 MB 推奨）
③ ダウンロードしたファイルサイズが 12.0 MB ちょうどか確認（中断時はファイルマネージャで削除して再ダウンロード）
④ 「設定 → セキュリティ → 提供元不明のアプリ」の許可を確認
