# タイトル / メインメニュー（TitleScreen）

## 機能概要

アプリ起動直後に表示されるトップ画面。各モードや設定への入口を提供する。
背景に選択中キャラの立ち絵プレビューと簡易アニメーションを表示。

## 画面要素

- タイトルロゴ表示（`title_logo`）
- 「はじめる」ボタン（`title_start_button`）
- 「キャラクター選択」ボタン（`title_character_select_button`）
- 「ランキング」ボタン（`title_ranking_button`）
- 「チュートリアル」ボタン（`title_tutorial_button`）
- 「設定」ボタン（`title_settings_button`）
- 背景アニメーション・選択中キャラ立ち絵プレビュー（`title_character_preview`）
- 画面ルート（`title_root`）

## ユーザー操作

- 「はじめる」押下 → `mode_select` へ遷移
- 「キャラクター選択」押下 → `character_select` へ遷移
- 「ランキング」押下 → `ranking?mode=endless` へ遷移（既定は endless）
- 「チュートリアル」押下 → `tutorial` へ遷移
- 「設定」押下 → `settings` へ遷移

## エラーケース

- 初回起動でセーブデータ未生成: 自動的に空 `AutoSaveEntity(id=0)` を生成して進行可能にする（CharacterRepository 初回シードと並走）。
- 画面回転・バックグラウンド復帰で BGM 二重再生: `AudioManager` 側で sceneId が同じなら再起動しない。

## データモデル

- `TitleUiState(selectedCharacter: Character?)`
- 起動時に `CharacterRepository.observe()` から先頭 selected 取得

## 受け入れ条件

- Given: アプリ初回起動
- When: タイトル画面が表示される
- Then: すべての主要ボタン（5 個）が押下可能で、各画面への遷移が成功する（Maestro smoke で 5 ルート遷移を検証）。
