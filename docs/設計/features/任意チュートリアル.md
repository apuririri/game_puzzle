# 任意チュートリアル（TutorialScreen）

## 機能概要

メインメニューから任意のタイミングで再生可能なチュートリアル。基本操作・連鎖の組み方を、サンプルプレイ＋字幕＋キャラクターボイスで実演する。初回起動時に強制はしない。

## 画面要素

- 画面ルート（`tutorial_root`）
- チュートリアル本文表示領域（`tutorial_text_area`）
- 操作デモ表示領域（`tutorial_demo_area`）
- 「次へ」ボタン（`tutorial_next_button`）
- 「とばす」ボタン（`tutorial_skip_button`）
- ページインジケータ（`tutorial_page_indicator`）

## ユーザー操作

- 「次へ」押下 → 次ページへ進む（最終ページ次は完了 → タイトルへ）
- 「とばす」押下 → タイトル画面へ戻る
- 全ページ閲覧完了 → 完了メッセージ → タイトル画面へ戻る

## エラーケース

- 途中でホームに戻り再開: DataStore の `tutorial.lastViewedPage` を読み、進行ページを復元。
- ボイス再生中に画面回転: ViewModel が rememberSaveable + AudioManager の継続再生で停止しない。

## データモデル

- 全 5 ページ（操作 / 連鎖の仕組み / 落下スピード / ハードドロップ / 大連鎖）
- 各ページに `title:String` `body:String` `voiceEventId:String` `demoAnimation:enum`

## 受け入れ条件

- Given: タイトル画面
- When: 「チュートリアル」ボタンを押下し最後まで進める
- Then: 操作デモ・字幕・ボイスが順次再生され、完了後にタイトル画面に戻る（Maestro flow `tutorial.yaml` で次へを 5 回タップして title_root に戻ることを確認）。
