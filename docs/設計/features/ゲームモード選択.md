# ゲームモード選択（ModeSelectScreen）

## 機能概要

タイトル画面の「はじめる」から遷移する、4 つのゲームモードへの入口。

## 画面要素

- 画面ルート（`mode_select_root`）
- 「エンドレス」ボタン（`mode_endless_button`）
- 「スコアアタック」ボタン（`mode_scoreattack_button`）
- 「ストーリー」ボタン（`mode_story_button`）
- 「CPU 対戦」ボタン（`mode_cpu_button`）
- 各モード説明テキスト（`mode_description_text`）
- 「戻る」ボタン（`mode_back_button`）

## ユーザー操作

- 各モードボタン押下 → 該当モードのプレイ画面（またはモード固有設定）へ遷移
  - エンドレス → `play/endless`
  - スコアアタック → `play/score_attack`
  - ストーリー → `play/story?characterId=<selected>&chapter=1`（または進行中の章）
  - CPU 対戦 → `play/cpu?difficulty=<settings.difficulty>`
- 「戻る」押下 → タイトル画面へ戻る

## エラーケース

- ストーリー進行状況が破損（パース失敗）: 章 1 から再開可能にし、警告ダイアログ `mode_story_warning_dialog` を表示。

## データモデル

- `ModeSelectUiState(storyResumeChapter: Int?)`

## 受け入れ条件

- Given: ゲームモード選択画面
- When: 4 つのモードボタンをそれぞれ押下
- Then: それぞれ対応するモードのプレイ画面へ遷移する（Maestro で 4 ルートを検証）。
