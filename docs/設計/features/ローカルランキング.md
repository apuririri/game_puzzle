# ローカルランキング（RankingScreen / TOP10）

## 機能概要

エンドレス／スコアアタック／CPU 対戦の各モード別に、端末内でハイスコア TOP10 を保持する。

## 画面要素

- 画面ルート（`ranking_root`）
- モード切り替えタブ
  - `ranking_tab_endless`
  - `ranking_tab_scoreattack`
  - `ranking_tab_cpu`
- ランキング一覧（`ranking_list`）
- 各行（`ranking_row_{rank}`、rank ∈ 1..10）: 順位 / スコア / 最大連鎖数 / 使用キャラ / 日付
- 「戻る」ボタン（`ranking_back_button`）
- データ未登録時メッセージ（`ranking_empty_text`）

## ユーザー操作

- タイトル → 「ランキング」押下 → ランキング画面表示
- モードタブ切り替えで各モードの TOP10 を表示
- リザルト画面で TOP10 入りした場合は強調表示（呼び出し側 ResultScreen の責務）

## エラーケース

- データ未登録: 「まだ記録がありません」を `ranking_empty_text` で表示
- スコア同点: 達成日時が古い順に上位表示（SQL `ORDER BY score DESC, playedAt ASC LIMIT 10`）

## データモデル

```kotlin
class RankingRepository(dao: HighScoreDao) {
  fun observeTop10(mode: String): Flow<List<HighScoreEntity>>
  suspend fun insertIfTop10(mode: String, score: Long, maxChain: Int, characterId: String, playedAt: Long): Int? // returns rank
}
```

## 受け入れ条件

- Given: エンドレスモードでハイスコア TOP10 入りを達成
- When: タイトル → ランキング画面を開く
- Then: 当該スコアがエンドレスタブの該当順位に表示される（Compose UI Test + DAO fixture）。
