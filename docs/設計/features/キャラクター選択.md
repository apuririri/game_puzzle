# キャラクター選択（CharacterSelectScreen）

## 機能概要

プレイに使用する美少女キャラクターを選ぶ画面。初回起動時から全 6 キャラが選択可能（v0.2 で「あぷりりり」追加）。
選択中キャラはタイトル・各モードに反映される（DataStore `selected.characterId`）。

## 画面要素

- 画面ルート（`character_select_root`）
- キャラクター一覧（横スクロール / `character_list`）
- 各キャラカード（`character_card_{characterId}`、id ∈ {hina, airi, yuki, mio, rin, apuririri}）
- 選択中キャラ大プレビュー（`character_preview_image`）
- ボイスプレビュー再生ボタン（`character_voice_preview_button`）
- 「決定」ボタン（`character_confirm_button`）

## ユーザー操作

- キャラカードをタップ → プレビューエリアが切り替わり、サンプルボイス（`voice/<id>/select_preview.ogg`）が 1 回再生
- 「決定」押下 → 選択キャラを DataStore に保存しタイトル画面へ戻る

## エラーケース

- 立ち絵画像の読み込みに失敗: プレースホルダ画像（grey + キャラ名テキスト）を表示し、エラー音は出さない。
- ボイス再生中に別キャラを選択: 直前のボイスを停止して新ボイスを再生（AudioManager 仕様）。

## データモデル

- `CharacterSelectUiState(characters: List<Character>, selectedId: String, isPreviewing: Boolean)`

## 受け入れ条件

- Given: キャラクター選択画面
- When: いずれかのキャラを選び「決定」を押下
- Then: 選択結果が DataStore に永続化され、再起動後も同じキャラが選択状態である（Compose UI Test + DataStore 観測）。
