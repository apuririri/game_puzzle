# スクリーンショット共有（共通機能 / 各画面のシェアボタン）

## 機能概要

リザルト画面・任意のタイミングで現在画面のスクリーンショットを取得し、OS 標準シェアシートで SNS や保存先に送る最小限のシェア機能。
本ファイルは「画面側にシェアボタンが配置される」要件を集約する（実装の中核は 共通基盤_ScreenCapture）。

## 画面要素

- リザルト画面の `share_button`（= `result_share_button` を `share_button` 共通名でも参照可能にする）
- プレイ画面の一時停止メニュー内 `share_button`（`playfield_pause_menu_share_button`）

## ユーザー操作

- 「シェア」押下 → 現在画面をキャプチャ → OS 標準シェアシート起動

## エラーケース

- View が描画途中: キャプチャ完了まで遅延し、失敗時は再試行案内 Toast
- 共有先アプリ未インストール: OS シェアシートの標準挙動に委ねる

## データモデル

該当なし（共通基盤_ScreenCapture を呼ぶだけ）。

## 受け入れ条件

- Given: リザルト画面表示中
- When: 「シェア」ボタンを押下
- Then: OS のシェアシートに当該スクリーンショットが添付された状態で表示される（Compose UI Test の Intents.intended で `ACTION_SEND` + `EXTRA_STREAM` を検証）。

## testTag 一覧
