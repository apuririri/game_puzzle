# エンドレスモード（EndlessScreen）

## 機能概要

ゲームオーバーになるまで延々と続く基本モード。スコアと連鎖数を競う。
本機能の Composable と ViewModel は他プレイモード（スコアアタック / CPU / ストーリーのバトルパート）から再利用可能な形で実装する。

## 画面要素

- 画面ルート（`endless_root`）
- プレイフィールド 6×14（`playfield_grid`）
- 落下中ぷよ（`playfield_current_pair`）
- ネクスト・ネクストネクスト表示（`playfield_next` / `playfield_next_next`）
- スコア表示（`playfield_score`）
- 連鎖数表示（`playfield_chain_count`）
- 一時停止ボタン（`playfield_pause_button`）
- 一時停止メニュー（`playfield_pause_menu`）
- 操作 UI: 左移動 / 右移動 / 回転 / 高速落下 / ハードドロップ
  - `playfield_btn_left` / `playfield_btn_right` / `playfield_btn_rotate` / `playfield_btn_softdrop` / `playfield_btn_harddrop`
- 連鎖演出オーバーレイ（共通基盤_ChainEffectRenderer の `chain_*` 群）

## ユーザー操作

- 左／右で 1 マス移動
- 回転で時計回り 90 度
- 高速落下押下で落下速度を上げる、離すと通常速度に戻る
- ハードドロップで即着地
- 3 つ以上の同色ぷよが縦横で隣接 → 消去 → 上のぷよが落下 → 連鎖判定（再帰）
- 一時停止ボタン押下 → メニュー（再開 / セーブ / ロード / シェア / タイトルへ）

## エラーケース

- 一時停止中にバックグラウンド復帰: 一時停止状態を維持
- プロセス再起動: 直近のオートセーブから「中断中」として復帰可能（リジューム選択可）
- 端末スリープ: ゲーム進行を停止し、復帰時に一時停止画面を表示

## データモデル

- `PlayUiState(field, currentPair, next, next2, score, chainCount, isPaused, isGameOver, mode="endless")`
- `Tick` を `withFrameNanos` で生成し、ChainEngine.step を駆動
- 着地ごとに SaveLoadManager.autoSave を呼ぶ

## 受け入れ条件

- Given: エンドレスモード開始直後
- When: 同色ぷよ 3 つを縦横で隣接させる
- Then: 当該ぷよが消去され、連鎖カウントが増加しスコアが加算される（Compose UI Test で 3 個積み → testTag `playfield_chain_count` のテキスト変化を検証）。
- Given: プレイ中にホーム退避 → kill → 再起動
- When: タイトル → 「はじめる」 → エンドレス
- Then: 「中断中」ダイアログから直前状態へ復帰可能（Maestro `endless_resume.yaml`）。
