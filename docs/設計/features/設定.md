# 設定（SettingsScreen）

## 機能概要

ゲーム全体の挙動を制御する設定画面（要件 §機能: 設定）。各設定変更は DataStore に即時反映され、全画面で次回再生から適用される。

## 画面要素

- 難易度選択（`settings_difficulty_select` / RadioGroup: Easy / Normal / Hard / Expert）
- BGM 音量スライダ（`settings_bgm_volume`）
- SE 音量スライダ（`settings_se_volume`）
- ボイス音量スライダ（`settings_voice_volume`）
- BGM ON/OFF トグル（`settings_bgm_toggle`）
- SE ON/OFF トグル（`settings_se_toggle`）
- ボイス ON/OFF トグル（`settings_voice_toggle`）
- 大連鎖クリップ自動保存 ON/OFF（`settings_chain_clip_toggle`）
- 「戻る」ボタン（`settings_back_button`）
- 画面ルート（`settings_root`）

## ユーザー操作

- 各設定変更 → DataStore に即時反映 → 全画面で次回再生から適用
- 難易度はエンドレス／スコアアタック／CPU 対戦のロジックパラメータに反映
- 「戻る」押下 → 直前画面へ `popBackStack()`

## エラーケース

- DataStore 書き込み失敗: スナックバー `settings_error_snackbar`「設定の保存に失敗しました」を表示し、UI 状態を直前値にロールバック。

## データモデル

`AppSettings`（共通基盤_SettingsRepository）を `Flow` で `collectAsState`、ユーザー操作で `update*` を呼ぶ。

## 受け入れ条件

- Given: 設定画面で BGM を OFF に変更
- When: タイトル画面に戻り、ゲームモードへ進む
- Then: BGM が再生されない（AudioManager 経由で抑止 / Compose UI Test + AudioManager fake で検証）。
