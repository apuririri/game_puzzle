# 音声再生統合（キャラクターボイス・BGM・SE 各画面統合）

## 機能概要

要件「機能: キャラクターボイス・BGM・SE 再生」の各画面側統合を集約する。
画面遷移・ぷよ操作・連鎖イベント・勝敗判定の各場面で適切な AudioCue を AudioManager に投げる。

## 画面要素

なし（共通機能。AudioManager は画面を持たない）。

## ユーザー操作

なし（画面遷移・ぷよ操作・連鎖イベント等が間接的に発火）。

## エラーケース

- ファイル欠落: 該当再生をスキップ、AppLog::w に警告のみ
- 通話着信: AudioFocus を譲渡し、復帰時に BGM のみ再生再開（SE/ボイスは再開しない）
- イヤホン接続/切断: AudioManager のフォーカスイベントで一時停止 → 復帰

## データモデル

シーン別 BGM マップ:
| 画面 / シーン | BGM sceneId |
|---|---|
| Title | `title` |
| ModeSelect | `mode_select` |
| Story (会話) | `story_dialog` |
| Play (通常) | `play_normal` |
| Play (5+連鎖中) | `play_intense` |
| Result (勝利) | `result_win` |
| Result (敗北) | `result_lose` |

操作 SE 対応:
| イベント | SE eventId |
|---|---|
| ボタンタップ | `button_tap` |
| 左/右移動 | `move` |
| 回転 | `rotate` |
| ハードドロップ | `harddrop` |
| 着地 | `land` |
| 連鎖小 | `pop_small` |
| 連鎖大 | `pop_big` |
| クリップ保存 | `clip_saved` |
| エラー | `error` |

ボイス対応（selectedCharacterId）:
| イベント | voice eventId |
|---|---|
| キャラ選択プレビュー | `select_preview` |
| 1〜2 連鎖 | `chain_1` / `chain_2` |
| 5+ 連鎖 | `chain_big` |
| 勝利 | `win` |
| 敗北 | `lose` |
| アイドル（タイトル長居） | `idle` |

## 受け入れ条件

- Given: 設定でボイス ON、BGM ON、SE ON
- When: プレイ中にぷよを操作・連鎖を発生させる
- Then: 操作 SE・連鎖ボイス・連鎖 SE・BGM が混ざって正しく再生される（fake AudioManager で SoundPool / MediaPlayer の play 呼び出し列を検証）。
