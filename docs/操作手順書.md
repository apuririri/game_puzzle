# 美少女連鎖パズル 操作手順書

## 1. 起動

アプリアイコン（美少女連鎖パズル）をタップするとタイトル画面が表示されます。

## 2. タイトル画面

5 つのメインボタン:

| ボタン | 動作 |
|---|---|
| はじめる | モード選択画面へ進みます |
| キャラクター選択 | 6 体のキャラから選択 |
| ランキング | 端末内ローカル TOP10 を表示 |
| チュートリアル | 基本操作を 5 ページで案内 |
| 設定 | 難易度・音量・ON/OFF を変更 |

選択中のキャラ名がタイトル下に表示されます。

## 3. キャラクター選択

- 横スクロールのカードからキャラを選択
- 「ボイス プレビュー」でサンプルボイスを試聴
- 「決定」を押すと選択を保存しタイトル画面に戻ります
- 選択結果は再起動後も保持されます

## 4. ゲームモード

| モード | 内容 |
|---|---|
| エンドレス | ゲームオーバーまで延々と続く基本モード |
| スコアアタック | 3 分間でハイスコアを競う |
| ストーリー | 章ごとに会話 → バトル → 会話を繰り返す |
| CPU 対戦 | CPU と 1 対 1 連鎖バトル |

## 5. プレイ操作

| 操作 | 説明 |
|---|---|
| ← / → | 落下中ぷよを 1 マス移動 |
| 回転 | 時計回りに 90 度回転 |
| 早落 | 高速落下 |
| 即着 | ハードドロップ（即着地） |
| 停止 / 再開 | ゲームを一時停止 |

同色ぷよを 3 つ以上隣接させると消去され、連鎖でスコアと連鎖数が加算されます。
5 連鎖以上で派手な演出 + 動画自動保存（設定で OFF 可）。

## 6. リザルト画面

ゲーム終了後に表示されます:

- スコア / 最大連鎖 / 使用キャラ
- TOP 10 入り時は強調表示
- 「シェア」で画面キャプチャを OS シェアシートに渡せます
- 「もう一度」で同モード再開、「タイトルへ」で戻る

## 7. ランキング画面

タイトル → ランキング で開きます。ENDLESS / SA / CPU のタブで切替。
データ未登録時は「まだ記録がありません」と表示されます。

## 8. 設定画面

- 難易度: Easy / Normal / Hard / Expert
- BGM / SE / ボイス: 音量スライダ + ON/OFF トグル
- 大連鎖クリップ自動保存: 5 連鎖以上の動画自動保存を有効化（要ストレージ容量）
- 設定は即時反映され、アプリ再起動後も保持されます

## 9. チュートリアル

任意で再生可能。「次へ」で進み、「とばす」で即終了。
途中で閉じても次回は最後のページから再開できます。

## 10. ストーリーモード

- 会話パートで「次へ」を押して台詞を進める
- バトル開始後はエンドレスと同じ操作
- 章クリア条件（スコア 1000）達成で進行状況を保存

## 11. 中断・再開

- プレイ中にホームに退避 → アプリ再起動で「直前の状態」から再開可能
- セーブスロット画面で複数スロットの管理が可能（実装中）

## 12. シェア

- リザルト画面の「シェア」ボタン
- プレイ中の一時停止メニューにも配置（実装中）
- スクリーンショットが Android 標準シェアシートに添付されます
