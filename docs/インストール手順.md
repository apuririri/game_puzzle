# 美少女連鎖パズル インストール手順（Android 実機）

## 前提

- Android 8.0（API 26）以上の端末
- USB ケーブル（PC ↔ スマホ）
- 開発者オプション + USB デバッグ有効化（後述）

## 1. 端末側の準備

1. 「設定」→「デバイス情報」→「ビルド番号」を 7 回タップ
2. 「開発者向けオプション」→「USB デバッグ」を ON
3. USB で PC に接続。最初の接続時、PC からの USB デバッグを「許可」

## 2. APK 入手

リリース APK は以下のいずれかから取得します:

- **開発者から直接受領**: `美少女連鎖パズル-<version>-release.apk`
- **本プロジェクトで自前ビルド**:
  ```bash
  bash autodev/scripts/build.sh release
  # 出力: app/build/outputs/apk/release/app-release.apk
  ```

## 3. インストール（adb）

```bash
# PC 側で adb が認識されていることを確認
adb devices
# List of devices attached
# <SERIAL>  device

# インストール
adb install -r 美少女連鎖パズル-<version>-release.apk
# Performing Streamed Install
# Success
```

## 4. 起動確認

ホーム画面または app drawer に「美少女連鎖パズル」アイコンが追加されています。
タップしてタイトル画面が表示されれば成功です。

## 5. アンインストール

```bash
adb uninstall com.example.myapp
```

または端末側で長押し → アンインストール。
アンインストール時はランキング・セーブデータも削除されます。

## 6. トラブルシューティング

| 症状 | 原因と対処 |
|---|---|
| `adb: command not found` | Android SDK の platform-tools が未導入。`brew install android-platform-tools` 等で導入 |
| `device unauthorized` | 端末側で USB デバッグ許可ダイアログが出ているはず。許可してください |
| `INSTALL_FAILED_VERSION_DOWNGRADE` | 既存バージョンが新しい。先に `adb uninstall com.example.myapp`（データ消えます） |
| `INSTALL_FAILED_UPDATE_INCOMPATIBLE` | 署名不一致。同上 |
| 起動直後にクラッシュ | logcat 確認: `adb logcat -d -t 200 \| grep -E "AppLog|AndroidRuntime"` |

## 7. アセットについて

本リリースの画像・音声はプレースホルダー（単色 + キャラ名 / 440Hz 短音）です。
正規アセットへの差替えは納品時に別途実施されます（要件 §3 / S1 Q5 参照）。
